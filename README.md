# AI Data Science Copilot

An AI-powered assistant that helps users analyze datasets, generate insights, and understand machine learning results using LLMs.

## Features
- Upload datasets
- Automated data analysis
- Generate AI-based insights
- Data visualization
- ML result explanation

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- Groq LLM
- Generative AI

## Setup

Clone repository

pip install -r requirements.txt

Create .env file:

GROQ_API_KEY=your_api_key

Run:

streamlit run app.py
